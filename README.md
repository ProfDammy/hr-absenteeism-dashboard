# HR Absenteeism Dashboard

This project analyzes employee absenteeism using Power BI.

## Objective
To identify patterns and causes of employee absenteeism and provide insights that support workforce management decisions.

## Tools Used
- Power BI
- Excel Dataset
- Data Modeling
- DAX Calculations

## Dashboard Features
- Total Absenteeism Hours
- Absence by Season
- Absence by Reason
- Absence by Weekday
- Monthly Absence Trends
- Employee Characteristics (BMI, Education Level)

## Key Insights
- Medical reasons account for the majority of absenteeism
- Spring recorded the highest number of absences
- Monday has the highest absenteeism among weekdays
- Workload peaks correspond with increased absence levels
